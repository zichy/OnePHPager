# 📝 OnePHPager

__OnePHPager__ makes a single page editable in the browser.

(You actually do not want to use this software since I don’t know how to write PHP.)

## Usage
1. Change the password in `index.php`. You may rename the file to `anything.php`. 
2. Upload the file to a server.
3. Go to `?admin` and login using your password.
4. Enter any HTML and submit.
5. Your content will be saved into `anything.txt` and served at `anything.php`.
